# 要件定義 - 横浜橋商店街 公式HP

## 目的
横浜橋商店街の公式HPを新規作成。情報発信・集客強化。商店街スタッフがお知らせを自分で更新できること。

## ページ構成
1. **トップ** - ヒーロー、お知らせ最新5件、各ページバナー、SNSリンク、関連HPリンク
2. **お店情報** - 約120店舗一覧（写真・店舗名・業態・営業時間・電話番号）、業種フィルタ、Google Map埋め込み
3. **アクセス** - Google Map埋め込み、最寄駅からの動線（写真付き）
4. **お問い合わせ** - Contact Form 7 フォーム + 電話番号・メール記載
5. **商店街について** - 歴史・概要・運営組織情報
6. **お知らせ一覧/個別** - WP標準投稿、カテゴリ分類、ページネーション

## 機能要件
- 店舗情報: カスタム投稿タイプ + ACF + 業種タクソノミー
- お知らせ: WP標準投稿、ブロックエディタで商店街スタッフが更新
- レスポンシブ: モバイルファースト（768px / 1024px）
- SEO: Yoast SEO、OGP対応、セマンティックHTML
- セキュリティ: SiteGuard、エスケープ処理、SSL

## デザイン
- カラー: 白 / 赤(#D32F2F) / 青(#1565C0)
- シンプルで見やすい。参考: https://www.togoshiginza.jp/

## 店舗データ
- 先方からExcelで提供 → CSV変換 → WP All Import で一括投入
- 約120店舗分
