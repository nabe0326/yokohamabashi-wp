# 運用マニュアル - 横浜橋商店街 公式HP

## 初期セットアップ

### 1. 必須プラグインのインストール

管理画面 → プラグイン → 新規追加 から以下をインストール・有効化してください。

| プラグイン名 | 用途 | 必須 |
|-------------|------|------|
| Advanced Custom Fields (ACF) | 店舗の詳細情報管理 | ○ |
| Contact Form 7 | お問い合わせフォーム | ○ |

### 2. パーマリンク設定

管理画面 → 設定 → パーマリンク → 「投稿名」を選択 → 変更を保存

※ これを行わないと店舗ページのURLが正しく動作しません。

### 3. 固定ページの作成

以下の固定ページを作成してください（スラッグが重要）。

| ページタイトル | スラッグ | 内容 |
|---------------|---------|------|
| アクセス | `access` | テンプレートが自動適用されます |
| 商店街について | `about` | テンプレートが自動適用されます |
| お問い合わせ | `contact` | テンプレートが自動適用されます |

**作成手順:**
1. 管理画面 → 固定ページ → 新規追加
2. タイトルを入力
3. 右サイドバー「URL」をクリック → スラッグを上記の通り設定
4. 公開

### 4. お問い合わせフォームの設定

1. 管理画面 → お問い合わせ → 新規追加
2. フォーム名を「お問い合わせ」に設定
3. 必要なフィールドを設定して保存
4. ショートコードをコピー（例: `[contact-form-7 id="123" title="お問い合わせ"]`）
5. `page-contact.php` の該当箇所を修正するか、管理画面でショートコードIDを確認

## 店舗情報の管理

### 店舗を追加する

1. 管理画面 → **店舗情報** → **新規店舗追加**
2. 以下の情報を入力:

| 項目 | 説明 | 必須 |
|------|------|------|
| タイトル | 店舗名 | ○ |
| 業種 | 右サイドバーから選択（複数可） | - |
| 営業時間 | 例: 10:00〜19:00 | ○ |
| 定休日 | 例: 毎週水曜日 | - |
| 電話番号 | 例: 045-123-4567 | - |
| 店舗写真 | メイン画像をアップロード | ○ |
| 住所 | 店舗の住所 | - |
| Google Map埋め込み | Google Mapのiframeコード | - |
| 一言紹介 | 店舗の説明文 | - |

3. **公開** をクリック

### Google Map埋め込みコードの取得方法

1. [Google Maps](https://www.google.co.jp/maps) で店舗を検索
2. 「共有」ボタンをクリック
3. 「地図を埋め込む」タブを選択
4. 「HTMLをコピー」をクリック
5. コピーしたコードを「Google Map埋め込み」フィールドに貼り付け

### 業種カテゴリの管理

管理画面 → 店舗情報 → **業種**

初期カテゴリ（必要に応じて追加）:
- 飲食
- 食品・生鮮
- 衣料
- 日用品・雑貨
- サービス
- 医療・薬局
- その他

## お知らせの管理

### お知らせを投稿する

1. 管理画面 → **投稿** → **新規追加**
2. タイトルと本文を入力
3. 必要に応じてアイキャッチ画像を設定
4. カテゴリを選択
5. **公開** をクリック

### お知らせカテゴリの管理

管理画面 → 投稿 → カテゴリ

例:
- イベント
- お休みのお知らせ
- キャンペーン
- その他

---

## ナビゲーションメニューの設定

1. 管理画面 → 外観 → メニュー
2. 「新しいメニューを作成」
3. メニュー名を入力（例: メインメニュー）
4. 固定ページや店舗一覧を追加
5. メニューの位置「primary」にチェック
6. メニューを保存

推奨メニュー構成:
- ホーム
- お知らせ
- 店舗一覧
- アクセス
- 商店街について
- お問い合わせ

---

## トラブルシューティング

### ページが404エラーになる

→ 管理画面 → 設定 → パーマリンク → 「変更を保存」をクリック

### 店舗の詳細情報が表示されない

→ ACFプラグインが有効化されているか確認

### お問い合わせフォームが表示されない

→ Contact Form 7プラグインが有効化されているか確認
→ ショートコードのIDが正しいか確認
